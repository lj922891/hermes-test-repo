# Hermes测试项目

## 项目目的
测试Hermes Agent的GitHub技能，包括：
- 仓库创建和管理
- 代码推送和拉取
- GitHub API访问
- 后续的代码审查和PR管理

## 文件说明
- `README.md` - 项目基本说明
- `create_repo.py` - 创建GitHub仓库的Python脚本
- `github_repo_info.json` - 仓库创建信息

## 使用技能
- github-auth: GitHub认证
- github-repo-management: 仓库管理
- github-code-review: 代码审查（后续）
- github-pr-workflow: PR工作流（后续）
